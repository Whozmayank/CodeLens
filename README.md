# 🔍 CodeLens — AI Code Explainer

CodeLens is an AI-powered web application that explains code in a simple, 
structured, and insightful way using Google's Gemini API.

## 🌐 Live Demo
🔗 https://codelens-beryl.vercel.app

---

## ✨ Features

- 🔐 **User Authentication** — Secure Register/Login with JWT
- 🤖 **AI Code Explanation** — Powered by Google Gemini API
- 📜 **Line-by-line Breakdown** — Understand exactly what each line does
- 📊 **Complexity Analysis** — Time & space complexity insights
- 💡 **Improvement Tips** — Get suggestions to write better code
- 🕘 **History Tracking** — All your past explanations saved per user
- 🗑️ **Delete History** — Clean up explanations you no longer need
- 🌙 **Dark Mode UI** — Easy on the eyes
- ⚡ **Fast & Responsive** — Works on all screen sizes
